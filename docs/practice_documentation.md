## 1. Общая информация о проекте

### Название проекта

Создание текстового редактора на Python с использованием Tkinter

### Цели и задачи проекта

**Цель**: Разработать полнофункциональный текстовый редактор на языке Python с графическим интерфейсом пользователя, изучив основы создания GUI-приложений и практики программирования.

**Задачи**:
- Изучить возможности модуля Tkinter для создания GUI-приложений
- Разработать текстовый редактор с базовыми функциями (создание, открытие, сохранение файлов)
- Реализовать дополнительные возможности (форматирование, темы, поиск)
- Подготовить техническую документацию и руководство

## 3. Теоретическая часть

### 3.1 Модуль Tkinter

Tkinter — стандартная библиотека Python для создания графического интерфейса пользователя (GUI). Она входит в состав стандартной библиотеки Python и не требует отдельной установки.

**Основные компоненты Tkinter**:

| Компонент | Описание |
|-----------|----------|
| Tk | Корневое окно приложения |
| Frame | Контейнер для размещения виджетов |
| Text | Многострочное текстовое поле |
| Entry | Однострочное текстовое поле |
| Button | Кнопка |
| Menu | Меню |
| Scrollbar | Полоса прокрутки |
| Label | Текстовая метка |

### 3.2 Архитектура приложения

Приложение построено по принципу объектно-ориентированного программирования. Основной класс `ModernTextEditor` инкапсулирует всю логику работы редактора.

**Ключевые классы и методы**:

```
ModernTextEditor
│
├── Инициализация и настройка
│   ├── __init__()            – создание окна, атрибутов, вызовы методов сборки
│   └── setup_colors()        – задание цветовых констант (светлая/тёмная тема, акцент)
│
├── Построение интерфейса
│   ├── create_menu_bar()     – меню: Файл, Правка, Вид
│   ├── create_toolbar()      – панель инструментов
│   ├── create_btn()          – фабрика кнопок
│   ├── create_text_area()    – текстовое поле tk.Text со скроллбарами, тегами форматирования
│   └── create_status_bar()   – строка состояния
│
├── Статус и обратная связь
│   ├── update_cursor()       – отображение текущей строки и столбца
│   └── update_status()       – временное сообщение, затем "Готов"
│
├── Форматирование текста (работа с тегами)
│   ├── make_bold()           – применение/снятие жирного
│   ├── make_italic()         – применение/снятие курсива
│   ├── make_underline()      – применение/снятие подчёркивания
│   ├── apply_format()        – общая toggle-логика для тегов bold/italic/underline
│   └── change_font()         – изменение гарнитуры/размера (для выделенного фрагмента или всего текста)
│
├── Работа с файлами
│   ├── open_file()           – открытие текстового файла (UTF-8)
│   ├── save_file()           – сохранение в файл
│   └── clear_text()          – очистка текстового поля
│
├── Система тем
│   ├── apply_theme()         – единый метод, перекрашивающий корневое окно, панель,
│   │                           текстовую область, кнопки, метки, разделители, стиль Combobox,
│   │                           а также вызывающий _set_menu_theme() для меню
│   ├── _set_menu_theme()     – рекурсивная перекраска всех пунктов меню
│   ├── light_theme()         – устанавливает dark_mode=False и вызывает apply_theme()
│   ├── dark_theme()          – устанавливает dark_mode=True  и вызывает apply_theme()
│   └── toggle_theme()        – переключает тему (кнопкой на панели инструментов)
│
├── Поиск и замена
│   ├── open_find()           – всплывающее окно для поиска
│   ├── open_replace()        – всплывающее окно для замены
│   ├── find_text()           – подсветка всех вхождений тегом "found"
│   ├── replace_text()        – глобальная замена через str.replace()
│   └── clear_search_highlight() – снятие жёлтой подсветки
│
└── Горячие клавиши
    └── bind_hotkeys()        – привязка Ctrl+O, Ctrl+S, Ctrl+F, Ctrl+H, Ctrl+B, Ctrl+I, Ctrl+U
```

## 4. Практическая реализация

### 4.1 Создание окна приложения

Корневое окно создаётся с помощью класса `Tk` из модуля `tkinter`. Устанавливаются параметры: заголовок, размер, минимальный размер.

### 4.2 Текстовая область

Для редактирования текста используется виджет `Text` с полосами прокрутки `Scrollbar`. Текстовая область поддерживает многострочный ввод, отмену действий (undo) и настраиваемое форматирование.

### 4.3 Меню и панель инструментов

Созданы:
- Меню "Файл" (Открыть, Сохранить, Очистить, Выход)
- Меню "Правка" (Найти, Заменить)
- Меню "Вид" (Светлая/Тёмная тема)
- Панель инструментов с кнопками быстрого доступа
- Выбор шрифта и размера через выпадающие списки

### 4.4 Форматирование текста

Реализовано форматирование с помощью тегов Tkinter:
- **Жирный** — тег "bold"
- **Курсив** — тег "italic"
- **Подчёркивание** — тег "underline"
- **Шрифт** — динамически создаваемые теги для каждого шрифта

### 4.5 Темы оформления

Реализованы две цветовые схемы:

**Светлая тема**:
- Фон: `#f0f0f0`
- Текст: `#2c3e50`
- Панель инструментов: `#e0e0e0`

**Тёмная тема**:
- Фон: `#2d2d2d`
- Текст: `#d4d4d4`
- Панель инструментов: `#3c3c3c`

### 4.6 Поиск и замена

- **Поиск**: диалоговое окно с подсветкой найденных совпадений
- **Замена**: возможность замены всех вхождений
- Подсветка результатов жёлтым цветом

### 4.7 Горячие клавиши

| Комбинация | Действие |
|------------|----------|
| Ctrl+O | Открыть файл |
| Ctrl+S | Сохранить файл |
| Ctrl+F | Найти |
| Ctrl+H | Заменить |
| Ctrl+B | Жирный шрифт |
| Ctrl+I | Курсив |
| Ctrl+U | Подчёркивание |
